# Integration Patterns

This document outlines high-level integration patterns for fintechs, banks, and payment companies evaluating en3.

## Patterns

- Server-to-platform API integration through documented boundaries.
- Client application integration through demo wallet interfaces.
- EVM network configuration through public chain templates.
- Webhook/event handling through placeholder contracts.

## Checklist

- Confirm wallet use case and user journey.
- Define required EVM-compatible networks.
- Define authentication and authorization model.
- Define transaction review and approval flow.
- Define reporting and event handling needs.
- Confirm private platform access process before production work.
